Your site's content is not moderated, or under workflow control, by default. It must be enabled for each content type separately. The good news is that it's really easy to do.

To enable moderation for a single content type, visit Structure » Content types. In the Operations drop-down list, click Manage moderation.

Check the Enable moderation states box, and you will see something like the following:

You must select at two moderation states from the Allowed moderation states checkboxes. Why two? Because all moderation states are designated as either published or non-published, and you must enable at least one of each. You will only be able to put content into one of the enabled states. For example, if you do not enable the "Needs review" state, you will not be able to apply that flag to content.

Finally, select a Default moderation state. This is the state that content will be in by default when you first create it.

And finally, press the Save button. You're all set!
